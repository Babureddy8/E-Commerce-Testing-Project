# E-Commerce Web Application Testing Project


## Overview
This project contains manual and automation testing scripts for an e-commerce web application.


## Features Tested
- Login
- Product Search
- Filters
- Add to Cart
- Checkout
- Order History


## Automation
- Selenium WebDriver scripts for Login, Search, and Add to Cart
- Page Object Model (POM) structure used
- Scripts written in Python


## Folder Structure
- Automation_Testing/scripts: Selenium test scripts
- Automation_Testing/utils: Helper classes
